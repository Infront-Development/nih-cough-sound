{% trans "PURPOSE:" %}
{% blocktrans %} In this study we will collect cough and breathing sound samples from healthy individuals, COVID-19 patients, and individuals with other infectious diseases. These samples will be used to develop an Artificial Intelligence (AI) model to distinguish the differences of diseases through cough and breathing sounds. Total number of participants is estimated to be 2400 for each type of disease. {% endblocktrans %}

{% trans "STUDY PROCEDURES:" %}
{% blocktrans %} Once you have agreed to participate in this study, you are required to answer few questions about yourself. After that, you are required to record and submit your cough and breathing sounds using the recording function through the website. The following are questions needed in this study: {% endblocktrans %}

  1. {% trans "Vaccination Status" %}
  2. {% trans "Respondent Group" %}
  3. {% trans "Date of Diagnosis" %}
  4. {% trans "Sex" %}
  5. {% trans "Age" %}
  6. {% trans "Medical Conditions" %}
  7. {% trans "Smoking Status" %}
  8. {% trans "Symptoms" %}
{% trans "The total participation time should be around 10 minutes." %}

{% blocktrans %} You will record your cough and breathing sounds using the recording capability in this website and the sound samples will be captured and sent to the study team for the purpose of research only.
{% endblocktrans %}

{% trans "RISK, PRECAUTIONS & POTENTIAL DISCOMFORT:" %}
{% blocktrans %} Make sure no other people are within 20 feet of you when recording your cough and breathing sounds. Make sure you are away from any source of noise to avoid unclear recordings. You may experience minimal discomfort while coughing, however, this will not cause any harm to you. {% endblocktrans %}

{% trans "POTENTIAL BENEFITS:" %}
{% blocktrans %} By participating in this research study, you will be contributing to a solution for a global problem. Eventually, your participation will contribute to the development of an automated infectious diseases screening web tool. {% endblocktrans %}

{% trans "CONFIDENTIALITY & PRIVACY:" %}
{% blocktrans %} Information from you will be limited to cough and breathing recordings and answers to the above questions. All data from the study will be stored in a secure local data server where they will be protected with encryption, and only accessible to participating researchers and organizations. We will keep a repository of the data after the study is completed so that we can use it for the development of similar technology for future use if another pandemic appears. {% endblocktrans %}
